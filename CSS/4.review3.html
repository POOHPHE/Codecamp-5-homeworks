4.2 grid และ flex แตกต่างกันอย่างไร
Flex ใช้เพื่อจัดการ layout 1D
Grid ใช้เพื่อจัดการ layout 2D

4.3 justify and align แตกต่างกันอย่างไร
Align ตัวหนังสือสร้างแต่ละบรรทัดให้ชิดกับขอบเว็บไซต์ด้านใดด้านหนึ่ง
Justify ตัวหนังสือที่อยู่อยู่ห่างจากขอบระยะเท่ากัน
